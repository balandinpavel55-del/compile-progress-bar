
# Compile Progress Bar VS Code Extension

This VS Code extension displays a progress bar while compiling your project. It simulates compile steps and visually indicates progress using VS Code's builtâ€‘in notification progress UI.

---
## Features
- Progress bar displayed in the VS Code notification area
- Fully customizable compile behavior
- Simple command to trigger compilation
- Works with any language or build tool (once you integrate real commands)


---
## Installation & Usage

### 1. Clone the Repository
```bash
git clone https://github.com/yourname/compile-progress-bar.git
cd compile-progress-bar
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Open in VS Code
```bash
code .
```

### 4. Run the Extension
- Press `F5` to start the Extension Host
- Open the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
- Run: **Start Compile With Progress Bar**
